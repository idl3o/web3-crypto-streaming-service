Using Panels in Vue Components

Vue Component Usage

The Panel UI is available as a reusable Vue component that accepts props for customization:

<PanelUI
  title="Dashboard Overview"
  variant="primary"
  :collapsible="true"
>
  <template #header>
    <div class="panel-header-actions">
      <button>Refresh</button>
    </div>
  </template>

  <div class="panel-content">
    Your content here...
  </div>

  <template #footer>
    <div class="panel-footer-actions">
      <button>View All</button>
    </div>
  </template>
</PanelUI>
