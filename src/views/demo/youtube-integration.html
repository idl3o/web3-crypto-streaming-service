YouTube Integration Example

This page demonstrates how to integrate YouTube videos into the Web3 Crypto Streaming Service platform.

Change Video

How to use: Enter a YouTube video ID (the part after v= in a YouTube URL) and click "Update Video" to change the embedded video.

Integration Code

To embed a YouTube video in your application, use the following code:

<div class="youtube-embed">
  <iframe 
    src="https://www.youtube.com/embed/VIDEO_ID?rel=0" 
    title="YouTube video player" 
    frameborder="0" 
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" 
    allowfullscreen></iframe>
</div>

Replace VIDEO_ID with the YouTube video ID you want to embed.
